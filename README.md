I started this project by sketching ideas, looking at two different designs, and combining what I liked from each into my own version.

The result is a fully functional shopping cart that dynamically displays products, lets users adjust quantities, and automatically updates the order total based on their selections.

Through this project, I practiced turning a design into a working application, handling user interactions, managing cart data, and calculating totals in real time. It was a great exercise in building a complete frontend feature from scratch. 

Although I learned a lot from this project, I hope to improve on my weaknesses and further strengthen my Tailwind CSS and JavaScript skills in my next personal project.

Technologies Used: Vanilla JavaScript, Tailwind CSS, and Vite.
